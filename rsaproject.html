RSA Project


The overall purpose of the program is to take a message from the user and encrypt it, using the given values.The user gives 3 values, E, N, and a text message. It is then encrypted and then an encrypted message is outputted. After, the user gives 3 values, D, N, and the encrypted message, it is then decrypted and the decrypted message is outputted. The inputs are the values and messages needed from the user in order for the program to work. The outputs are the encrypted message from the encrypt output and the decrypted message from the decrypted output.

The procedure takes 3 inputs from the user and uses commands, such as pow, to encrypt the message. It contributed to the overall functionality of the program as it is how messages get encrypted, which is one half of the purpose of the program. The algorithm works by taking the users inputs, and then it numerizes each character in the users input. Next, it uses the pow and char commands, along with the users inputs for e and n, to fully encrypt the message. Finally, the fully encrypted message is printed into the selected listbox.
