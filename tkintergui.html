Tkinter GUI


The overall purpose of the program is to convert US Dollar amounts to MX Pesos. The user inputs a number into the entry box, and when they press the convert button the number is converted into MX Pesos. The input is the users entry in the entry box, and the output is the converted number in the list box.

The list name is called "names". When the user saves the current values in the list box to my txt file named "value.txt". They can also open the list, which takes all the values from "value.txt" and inserts it into the list box. This list is important so that the user does not need to always have their values open, if this was not in the code my code would be written differently because I would need the values to be stored in a different way.
